How to Use These Examples

Follow these steps to implement the examples.

Hint: When customizing the JavaScript resolve the REST URL in your browser to understand what object names or paths to objects you will be defining.

1. Create the HTML

  • Start with a standard html page 
                            

<html>
    <head>
        <!-- meta tags for bootstrap -->
        <meta charset="utf-8">
        <meta name="viewport"
        content="width=device-width, initial-scale=1.0">
        <title>MyJSPage</title>
    </head>
    <body>

    </body>
</html>
  • Add css elements to the head of the page 
                            
<head>
    <title>MyJsPage</title>
    <link href="bootstrap3/css/bootstrap.css"
    rel="stylesheet">
    <link href="cts2/css/cts2.css" rel="stylesheet">
</head>
  • Add the example html snippet to the page body 
                            
<div class="btn-group">
    <a class="btn btn-primary" href="#">
    <i class="icon-user icon-white"></i>All Code Systems</a>
    <a class="btn btn-primary dropdown-toggle"
    data-toggle="dropdown" href="#">
        <span class="caret">
        </span>
    </a>
    <ul class="dropdown-menu" id="codesystems">
        <li><a href="#"><b>Code Systems</b></a></li>
        <li class="divider"></li>
    </ul>
</div>
  • Import the JavaScript files last to improve page loading performance. (Just before the closing body tag.) 
                            
<script
src="https://code.jquery.com/jquery-1.10.2.min.js">
</script>
<script src="bootstrap3/js/bootstrap.min.js"></script>
<script src="cts2/js/codeSystemDropDown.js"></script>

2. Get the Files

  • Clone the repository or download the zip file from the LexEVS CTS2 examples GitHub repository.
  • Move the needed file items to a new directory

3. Test the Code on the Resulting Page
